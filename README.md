# 👕 PyTorch ANN for Fashion-MNIST (Small)

A minimal PyTorch project that trains a simple artificial neural network (ANN) on the `FMNIST_small.csv` dataset (a reduced Fashion‑MNIST subset). Perfect for learning or quick prototyping.

## Dataset

- `FMNIST_small.csv`: each row = 784 pixel values (28×28 grayscale) + label (0–9)
- Labels: T‑shirt, Trouser, Pullover, Dress, Coat, Sandal, Shirt, Sneaker, Bag, Ankle boot

Place the CSV file inside a `data/` folder.

## Requirements

- Python 3.8+
- PyTorch
- Pandas, NumPy

Install everything with:
```bash
pip install torch pandas numpy scikit-learn matplotlib
